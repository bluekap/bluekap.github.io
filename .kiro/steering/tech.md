# Technology Stack

## Frontend Technologies
- **HTML5**: Semantic markup with modern standards
- **CSS3**: Custom styling with advanced features (flexbox, grid, animations)
- **Tailwind CSS**: Utility-first CSS framework (CDN version 2.2.19)
- **JavaScript (ES5/jQuery)**: Interactive functionality and animations
- **jQuery**: DOM manipulation and event handling (version 1.10.2)

## Key Libraries & Frameworks
- **Font Awesome**: Icon library (version 6.0.0)
- **jQuery Plugins**:
  - FitText: Responsive text scaling
  - FlexSlider: Image/content sliders
  - Waypoints: Scroll-triggered animations
  - Magnific Popup: Lightbox functionality
  - Modernizr: Feature detection

## Backend
- **PHP**: Server-side contact form processing
- **Native PHP Mail**: Email functionality for contact forms

## Development Approach
- **Mobile-first responsive design**
- **Progressive enhancement**
- **Vanilla JavaScript with jQuery for compatibility**
- **CSS custom properties and modern layout techniques**
- **Graceful degradation for older browsers**

## Build Process
This is a static website with no build system required. Files are served directly.

## Common Commands
- **Local Development**: Use any local server (e.g., `python -m http.server`, Live Server extension)
- **Deployment**: Upload files directly to web server
- **Testing**: Test contact form functionality requires PHP server environment

## Browser Support
- Modern browsers (Chrome, Firefox, Safari, Edge)
- IE8+ support through conditional comments and fallbacks
